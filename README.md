# 🏏 Virat Kohli Career Statistics Dashboard

## 1️⃣ Description
An interactive Power BI dashboard that provides a complete analysis of Virat Kohli’s cricket career across all formats (IPL, ODI, T20, Test). It helps visualize performance trends, consistency, and key milestones over time.

---

## 2️⃣ Short Description / Purpose
To analyze and present Virat Kohli’s career statistics in a visual and interactive way, enabling users to explore performance insights and trends easily.

---

## 3️⃣ Tech Stack
- 📊 Power BI (Dashboard & Visualization)
- 📁 Microsoft Excel (Data Processing)
- 🧠 DAX (Basic Calculations & KPIs)

---

## 4️⃣ Data Sources
- Public cricket statistics dataset (compiled from online cricket records)
- Structured and cleaned using Excel before visualization

---

## 5️⃣ Feature Highlights
- 📈 KPI Cards: Total Runs, Average, Strike Rate, Innings  
- 📊 Runs by Year analysis (performance trends)  
- 🥧 Boundaries breakdown (Fours vs Sixes)  
- 📉 Tons (100s & 50s) trend visualization  
- 🎯 Interactive filters:
  - Format (IPL, ODI, T20, Test)
  - Custom Year Range slicer  
- ⚡ Dynamic and user-friendly dashboard design
